# Divo Portfolio Pro

## Overview
A personal portfolio website for a video content creator, featuring a modern dark theme design with React and Vite.

## Tech Stack
- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS (via CDN)
- **Icons**: Lucide React

## Project Structure
```
.
├── App.tsx          # Main application component
├── index.tsx        # React entry point
├── index.html       # HTML template
├── constants.tsx    # Application constants/data
├── types.ts         # TypeScript type definitions
├── vite.config.ts   # Vite configuration
└── tsconfig.json    # TypeScript configuration
```

## Development
- Run `npm run dev` to start the development server on port 5000
- The app uses hot module replacement for fast development

## Build & Deployment
- Run `npm run build` to create a production build
- Output is generated in the `dist` folder
- Configured for static deployment
